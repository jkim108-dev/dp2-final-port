Modular
Bag System

A modular carrying system designed to support seamless transitions between work, fitness, and everyday mobility.

Overview
This project began with observing the inconvenience users face when carrying different bags for work, exercise, and daily life.

The design focuses on creating a modular structure that allows users to adapt the bag configuration depending on context and activity.

Design Process

01
Research
Investigated user behavior and carrying habits across work and lifestyle environments.
02
Development
Explored modular structures, material systems, and adaptable storage configurations.
03
Prototype
Built and refined physical prototypes to test usability, proportions, and fabrication methods.
